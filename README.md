# LeaveBy

Android application developed using Kotlin and OCR technology to scan work schedules, calculate hours worked, and automate leave-time alarms.

## Features
- OCR schedule scanning
- Automatic hour calculations
- Alarm notifications
- Overtime tracking
- Persistent settings and scheduling logic

## Technologies Used
- Kotlin
- Android Studio
- OCR / ML Kit

## Purpose
Built to solve a real-world scheduling and overtime tracking problem through automation and mobile development.
